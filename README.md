# 🚀 UniMan - The Ultimate University Management App (v1.2 Beta)

![Vanilla JS](https://img.shields.io/badge/JavaScript-Vanilla-yellow)
![UI/UX](https://img.shields.io/badge/UI%2FUX-Premium-success)

**UniMan** is a modern, beautifully designed web application built specifically for university students to manage their academic life effortlessly. It features a sleek, "App Store" quality user interface with dynamic color-coding, smooth animations, and local data persistence.

## ✨ Key Features

### 🧮 1. Smart Grade Calculator (Vize/Final)
Calculate exactly what you need to score on your final exam to pass a course or hit your target grade. 
* Supports custom mid-term and extra activity percentages.
* **Dynamic Feedback:** Analyzes your required score and gives emotional, color-coded feedback (e.g., "Grab a coffee ☕" for easy passes, or "You need a miracle 🚑" for impossible scores).
* Saves your courses with beautiful, color-coded target badges.

### 📅 2. Attendance Tracker (Devamsızlık)
Never fail a course due to absenteeism again.
* Set maximum absence limits for each course.
* Features a dynamic progress bar that turns from green to warning orange, and finally danger red as you approach your limit.
* One-click increment/decrement buttons.

### 🎓 3. GPA (AGNO) Calculator
Keep track of your semester average with ease.
* Enter ECTS (Credit) values and Letter Grades (AA to FF).
* **Traffic Light UI:** Letter grades are displayed using a beautiful color gradient system (Emerald Green for 'AA', descending to Danger Red for 'FF').
* Automatically calculates and updates your overall weighted GPA in real-time.

### ⏱️ 4. Pomodoro Focus Timer
Boost your studying productivity with the scientifically proven Pomodoro technique.
* 25-minute focus sessions with a sleek, animated SVG circular progress ring.
* Clean and distraction-free UI.
* Plays a satisfying chime alert when the session is complete to notify you it's time for a 5-minute break.

## 🛠️ Tech Stack

* **HTML5:** Semantic and structured markup.
* **CSS3:** Custom styling, flexbox layouts, smooth transitions, and dynamic SVG animations. (No external UI libraries used).
* **Vanilla JavaScript:** DOM manipulation, mathematical calculations, and timer logic.
* **LocalStorage API:** All user data (grades, attendance, courses) is saved directly to the browser, ensuring privacy and persistence without needing a database.
